# AURIX_Tricore_Development_Repository

Steps to follow to import a project:

Step 1: Install the Aurix Development Studio from infineon's website : https://www.infineon.com/cms/en/product/promopages/aurix-development-studio/

Step 3: Download the git repo or clone it.

Step 3: Click on import AURIX project -> Local Code Examples Repository -> Select the path to the Repository (Location where you downloaded the git repo). 

Done
